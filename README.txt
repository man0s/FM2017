Οδηγίες χρήσης:

1. Αν έχετε εγκατεστημένο το τελευταίο(έκδοση 8u144) Java SE Runtime Environment τότε αγνοείστε αυτό το βήμα, αλλίως θα πρέπει να το εγκαταστήσετε απο τον παρακάτω σύνδεσμο --> http://www.oracle.com/technetwork/java/javase/downloads/jre8-downloads-2133155.html

2. Μεταβείτε στον φάκελο Executables του τρέχοντος φακέλου και εκτελέστε το αρχείο FM2017.jar για να ξεκινήσει το πρόγραμμα.

3. Αρχικά θα πρέπει να μεταβείτε στις ρυθμήσεις της βάσης απο το μενού(Settings/Database Configuration). Έπειτα να αλλάξετε τα στοιχεία της βάσης με τα δικά σας. Τέλος, πατήστε το κουμπί "Connect". 

   Ως default τιμές της βάσης έχουν οριστεί οι ακόλουθες:
   IP: 127.0.0.1
   NAME: FM2017
   USERNAME: root
   PASSWORD: toor

4. Είστε έτοιμοί!
